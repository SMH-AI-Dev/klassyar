# Contributing to KlassYar

We love your input! We want to make contributing as easy and transparent as possible.

## Development Process

1. Fork the repo and create your branch from `main`.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Make your changes.
5. Ensure the app builds successfully: `npm run build`.

## Code Style

- We use modern React with hooks and functional components.
- Follow existing code patterns (RTL Persian support, clay-element classes).
- Use Tailwind CSS for styling — avoid inline styles.
- All UI text must support Persian (RTL).

## Pull Requests

1. Update the README.md with details of changes if needed.
2. Update the version in package.json following semver.
3. The PR will be merged once approved.

## Issues

Report bugs and request features via GitHub Issues.
