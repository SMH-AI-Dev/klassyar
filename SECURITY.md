# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in KlassYar, please report it by emailing klasyar.smh@gmail.com.

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days.
