## مراحل تست سیستم 🧪

### تست 1: ذخیره‌سازی بازی چرخ گردان
1. به صفحه Dashboard بروید
2. روی دکمه "فعالیت جدید" کلیک کنید
3. "چرخ گردان" را انتخاب کنید
4. موارد زیر را پر کنید:
   - عنوان: "تست چرخ گردان"
   - توضیحات: "این یک تست است"
   - گزینه‌ها: حداقل 4 گزینه اضافه کنید
5. روی "ذخیره فعالیت" کلیک کنید
6. ✅ باید به Dashboard برگردید و فعالیت جدید را ببینید

### تست 2: اجرای بازی
1. در Dashboard روی کارت فعالیت ایجاد شده کلیک کنید
2. روی دکمه "بازی" کلیک کنید
3. ✅ باید صفحه Play باز شود و چرخ نمایش داده شود
4. روی دکمه "چرخش چرخ" کلیک کنید
5. ✅ چرخ باید بچرخد و پس از 4 ثانیه نتیجه نمایش داده شود

### تست 3: بررسی localStorage
1. در مرورگر، کلید F12 را فشار دهید
2. به تب Application بروید
3. در بخش Storage > Local Storage، آیتم `klassyar_data` را باز کنید
4. ✅ باید داده‌های ذخیره شده را ببینید (activities, classes, results, user)

### تست 4: سایر بازی‌ها
برای اطمینان از عملکرد کامل سیستم، این بازی‌ها را هم تست کنید:
- ✅ Flashcard (کارت‌های آموزشی)
- ✅ Quiz (آزمون)
- ✅ Matching (جورکردنی)
- ✅ Memory Game (بازی حافظه)
- ✅ Group Sort (مرتب‌سازی گروهی)
- ✅ Wordsearch (پیدا کردن کلمه)

## عیب‌یابی مشکلات احتمالی 🔧

### مشکل: فعالیت ذخیره نمی‌شود
**راه‌حل:**
```javascript
// در Console مرورگر این کد را اجرا کنید:
localStorage.clear();
// سپس صفحه را رفرش کنید (Ctrl+F5)
```

### مشکل: صفحه Play باز نمی‌شود
**بررسی کنید:**
1. آیا نام فایل Play درست است؟ (مثلاً `PlayRandom_wheel.jsx`)
2. آیا Route در فایل router اضافه شده؟
3. در Console خطایی وجود دارد؟

### مشکل: چرخ نمی‌چرخد
**بررسی کنید:**
1. آیا گزینه‌ها در content.items ذخیره شده‌اند؟
2. در Console خطای JavaScript وجود دارد؟
3. CSS transition به درستی اعمال شده؟

```javascript
  description: "توضیحات",

  content: {
    items: ["گزینه 1", "گزینه 2", "گزینه 3", "گزینه 4"]
  },
  created_date: "2024-01-01T00:00:00.000Z",
  created_by: "user@example.com",
  plays_count: 0

```

## نکات مهم ⚠️

1. **localStorage** محدودیت حجمی دارد (معمولاً 5-10MB)
2. داده‌ها در localStorage به صورت **دائم** ذخیره می‌شوند
3. برای پاک کردن همه داده‌ها: `localStorage.clear()`
4. برای backup: داده `klassyar_data` را کپی و در جای امنی ذخیره کنید
