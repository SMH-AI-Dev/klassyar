# دستورالعمل تست

## مشکل رفع شده:
✅ فعالیت‌ها اکنون با `created_by` ذخیره می‌شوند
✅ لینک‌های Play درست شده‌اند
✅ دکمه Edit اضافه شده
✅ دکمه Delete با تأیید اضافه شده

## نحوه تست:

### 1. ساخت فعالیت جدید:
1. روی دکمه "فعالیت جدید" کلیک کنید
2. یکی از بازی‌ها را انتخاب کنید (مثلاً Quiz)
3. عنوان و محتوا را وارد کنید
4. ذخیره کنید

### 2. بررسی نمایش:
- فعالیت باید در "فعالیت‌های شما" نمایش داده شود
- نام بازی باید صحیح باشد (فارسی)
- آیکون باید نمایش داده شود

### 3. تست بازی:
- روی دکمه "بازی" کلیک کنید
- بازی باید باز شود و اجرا شود

### 4. تست ویرایش:
- روی دکمه آبی Edit (✏️) کلیک کنید
- صفحه ویرایش باید باز شود

### 5. تست حذف:
- روی دکمه قرمز Delete (🗑️) کلیک کنید
- پیام تأیید باید نمایش داده شود
- پس از تأیید، فعالیت حذف شود

## URL Pattern:
- Play: `/Play?type=quiz&id=xxx`
- Edit: `/Edit?type=quiz&id=xxx`

## فیلترینگ در Dashboard:
```javascript
base44.entities.Activity.filter({ created_by: me.email })
```

این فقط فعالیت‌های کاربر فعلی را نمایش می‌دهد.
